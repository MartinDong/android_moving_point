# android_moving_point
移动点的轨迹动态展示
## 前述 ##
- [高德官网申请Key](http://lbs.amap.com/dev/#/).
- 阅读[地图参考手册](http://a.amap.com/lbs/static/unzip/Android_Map_Doc/index.html).
